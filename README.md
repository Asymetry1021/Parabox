# Parabox
A clone for the game patrick's parabox.

## Getting Started

This repository is a work in progress for creating a clone of Patrick's Parabox, a puzzle game with recursive box mechanics.

## Adding Your Code

If you have existing code files that you want to add to this repository, please see the [CONTRIBUTING.md](CONTRIBUTING.md) file for detailed instructions on how to import your files.

## Quick Start for Contributors

1. Clone this repository
2. Add your files to the appropriate directories
3. Use `git add` to stage your files
4. Commit with `git commit -m "Your message"`
5. Push with `git push`

For detailed instructions, see [CONTRIBUTING.md](CONTRIBUTING.md).
